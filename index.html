泰拉瑞亚模组开发教程

基于 tModLoader 1.4.4，从第一个物品到发布上架 Steam Workshop 的完整指南

面向有 C# 基础的开发者 · 18 章系统课程 · 含完整代码示例

课程简介

泰拉瑞亚（Terraria）是一款拥有庞大玩家群体的 2D 沙盒游戏，而 tModLoader 是官方支持的模组框架， 让你可以用 C# 编写全新的游戏内容——武器、装备、NPC、Boss、生物群落，甚至是全新的游戏机制。

本教程假设你已经具备 C# 基础语法知识（类、继承、属性、方法重写），将带你从零开始掌握 tModLoader 模组开发的完整流程。 学完本课程后，你将能够独立开发、调试和发布一个完整的泰拉瑞亚模组。

环境要求：Steam 版泰拉瑞亚 + tModLoader（均可通过 Steam 免费获取 tModLoader）、Visual Studio 2022 或 VS Code + C# 扩展。

学习路线

建议按顺序学习。前 8 章是核心基础，9-14 章是中级进阶，15-18 章是高级主题。

推荐学习方式 这套教程最适合一边读一边做。建议你把前 8 章里的示例串成一个小型练手模组：先做一个材料和一把武器，再补弹幕、Buff、敌怪与 Boss。到了后半程，再把世界生成、生物群落、UI、联机和发布逐步接进去。这样你学到的不是孤立 API，而是一条真正能跑通的模组开发路线。
